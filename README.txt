CARA DEPLOY KE VERCEL

Pilihan paling langsung:
1. Pastikan Node.js sudah terpasang dari https://nodejs.org.
2. Klik dua kali deploy-vercel.bat.
3. Login ke Vercel ketika diminta.
4. Jawab pertanyaan konfigurasi dengan menekan Enter.
5. Setelah selesai, Vercel memberikan link seperti:
   https://website-tugas-chandra.vercel.app

Pilihan melalui GitHub:
1. Unggah semua file dalam folder ini ke repository GitHub.
2. Buka https://vercel.com dan login.
3. Klik Add New, pilih Project, lalu import repository tersebut.
4. Framework Preset pilih Other.
5. Root Directory biarkan ./ dan Build Command dikosongkan.
6. Klik Deploy.

CARA MENJALANKAN DI KOMPUTER

1. Pastikan Python sudah terpasang.
2. Klik dua kali file jalankan.bat.
3. Browser akan membuka http://localhost:8000.
4. Jangan tutup terminal selama website digunakan.

Tanpa localhost:
File index.html juga dapat dibuka langsung dengan klik dua kali.

Akses dari HP pada Wi-Fi yang sama:
1. Jalankan ipconfig pada Command Prompt.
2. Cari IPv4 Address komputer, misalnya 192.168.1.10.
3. Buka http://192.168.1.10:8000 pada HP.
